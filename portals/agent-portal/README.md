# Eazepay Agent Portal

React-based frontend for agent customer registration with biometric enrollment.

## Features

- ✅ Customer registration with all 10 fingers + palms
- ✅ Step-by-step biometric capture interface
- ✅ Primary finger designation
- ✅ Real-time capture status
- ✅ Review before submission
- ✅ Success confirmation with next steps

## Setup

```bash
cd portals/agent-portal
npm install
npm start
```

## Usage

1. Agent logs in
2. Navigates to "Register Customer"
3. Enters customer details
4. Captures all biometrics
5. Reviews and submits
6. Customer ready to pay!
